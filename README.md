# git-line

```sh
Usage: index [options] [command]

  Git command line tool


  Options:

    -V, --version  output the version number
    -h, --help     output usage information


  Commands:

    status|st         Select files to commit
    commit-push|cmp   Define a commit message
```

Installing globally:
```sh
npm install -g
```

Test app
```sh
npm link
```
